漫画品鉴

发表于 2025-01-24 更新于 2025-02-22

等不及了，快端上来罢！

2025年1月24日

今天在看b站up的漫画推荐时，突然意识到我也应该给自己看过的漫画归下档，以便之后的回顾和别的啥的。不过这里先叠甲，仅代表个人看法，甚至于仅代表个人当时看法，因为最近我回头看之前不大喜欢的汉化又发现真香人都是会变的。考虑到要整理的挺多，想到哪个就写上哪个，没有特定顺序。方便起见，5分制作评分吧。

阅读设备

主要是陪伴我多年的kindle paperwhite3，初三开始用它看漫画，最开始看的是火影？还是什么？总之已经成为我身体的一部分了。仔细一想我熬夜基本全是在看漫画，淦哦。

然后是我的平板三丧s8+，足够大的屏幕使阅读字较为友好。在kindle上看漫画导致我不是很喜欢看字，像当时看银魂纯意识流，字太多了，在kindle上看眼要瞎了，根本不看字只看画面（我讨厌轻改也是这个原因，字多）。原来使用tachiyomi，现在用mihon。

然后就是我的手机，用来追一些连载。

漫画

分类并不合理，像一本漫画肯定有诸多元素，仅仅凭我当时的第一印象进行分类。

搞笑

我个人比较喜欢搞笑漫画，我看的漫画也绝大多数都是这一类。甚至于如果有本漫画里没有搞笑元素，在我心里评分就会低一点这样。（图一乐反正就是）

魔王遭難中旁邊還有愉快的夥伴們唷

  1. 4/5 优秀
  2. 搞笑日常漫画
  3. #遠田マリモ #講談社 #別冊少年マガジン
  4. 魔王遭難中!!! 愉快な仲間達を添えて

日常搞笑，奇幻元素，甚至还有哥们最爱的冒险求生元素，太棒了很合口味。画风初看线条感觉有点乱，不过影响不大，没到会因为画风弃了的程度。但是问题是太短了，只有两卷！

异种族风俗娘评鉴指南

  1. 5/5 独秀！
  2. emmm，看名字也看得出来，搞笑节操漫画
  3. #masha 作画， #天原 原作
  4. 異種族レビュアーズ

很架空世界，我感觉作者有在注意细节和架空世界下的一些，emm，不同的生活发展走向？有点类似迷宫饭（有一说一，跟迷宫干相比，这个更像迷宫干吧）。不过更新很慢，很慢很慢。天原的另一本，33岁单身女骑士队长也是，更新得很慢很慢。可能因为架空世界需要构思，同时还需要保持好笑不是特别简单吧。

33歳独身女騎士隊長。

  1. 4/5 优秀
  2. 同上，搞笑节操漫画
  3. #天原
  4. 33歳独身女騎士隊長。

青年漫？（反正肯定不是少年漫），不适合在kindle或者手机上看，字太小。我是在平板上看到，阅读体验好太多。

貞操逆転世界

  1. 2/5 不是很喜欢
  2. 搞笑漫
  3. #天原

首先作画不是特别出彩，中规中矩的那种。然后感觉剧情拖沓。有一说一，看的漫画越来越多，很多漫画怎么样我早就不记得了。不过从哥们留有的印象来看，也就能拿个2分。

千绪的通学路

  1. 4/5 优秀
  2. 搞笑日常漫画
  3. #川崎直孝
  4. ちおちゃんの通学路

我记得之前在copy看过两三话来着，当时应该是不喜欢。现在一看感觉还不错。说实话看到偏后面就感觉节奏有点拖沓了。

点到为止

  1. 4/5 优秀
  2. 搞笑节操？漫画
  3. #川崎直孝
  4. おしまい

与千绪的通学路一个作者，带点节操元素。比较喜欢这种青年漫？（这算青年漫吗？）还有就是跟异种族风俗娘品鉴指南比较类似。

摄影部

  1. 4/5
  2. 作者说这是黄色摄影漫画

就搞笑日常漫画，带点节操元素，就3卷。另外跟摄影没啥关系。

不值一拍

  1. 5/5
  2. #二駅ずい

搞笑日常漫画，带节操元素。我写摄影部时想起来了，把这个也写了把。情节很搞，某种程度上算恋爱漫画？2025年2月22日，看到21话。很有意思，截至目前的话我愿意给5分。感觉全篇每一个正常人，很欢乐。

飛翔的魔女

  1. 5/5
  2. #石塚千寻

魔法，日常，搞笑。我很喜欢，很有意思，但是更新的是真慢！！

異世界幻想太！臭！了！

  1. 4/5
  2. #鬼頭サケル

很短，设定很有意思，确实没谁描绘过异世界怎么拉屎 。有一说一，我不知道是不是该把这个放到涩涩那个分类里。

冒险

感觉一些奇幻作品大部分也都可以分到这里奥

丁丁历险记

  1. 5/5 满分！神作
  2. 第一本丁丁在苏联画得很潦草简略，最后一本鲨鱼湖是动画且作者没参与，俩都不推荐。
  3. #埃尔热

volmoe上有人发布自己制作的切图横板，解决了在kindle阅读字太小的问题。上传者简直就是我的神！

史高治的光辉岁月

  1. 5/5 神作
  2. #迪士尼

这本还有外传我都是买的实体书看的。哥们真的很喜欢迪士尼的漫画。但是终极米迷那些书很容易开胶，最后都扔了。这本光辉岁月因为意外破损了，我小心翼翼地粘好的。

魔男伊奇

  1. 3/5
  2. #宇佐崎しろ #西修
  3. 狩猎魔法的漫画，奇幻

#西修 是入间同学入魔了的作者，这部好像不是她作画的。画风偏精美，然后跟入间同学一样穿插点搞笑元素，截止目前（2025年2月22日），我看到20话。我不确定这个，最后会不会跟入间同学一样，搞得节奏比较拖沓。但是从这前20话来看，还是比较好，剧情进展得比较流畅。

然后就是画风比较精美，故事也没啥硬伤，就看白，感觉一般化的作品。无功无过，比较一般的作品。

美食

酒之仄径

  1. 4/5
  2. #ラズウェル細木

我忌酒了，但是很喜欢看这个。画风偏简单，但是情节足够吸引人，让我想起来当时在漫画party上看的单伟画的有关吃的回忆漫画？每篇之后都有俳pai句，没咋仔细看过，不过翻译俳句估计挺难为译者的。

插一嘴，当时看漫画party时，猫小乐和单伟都在青岛（我一直以为猫小乐是青岛人，结果一看百科不是，吉林人。单伟是青岛人），当时也就感觉挺奇妙，谁曾想后来我也跑青岛念书去了。当时报志愿全是报的计算机和网安，但是报那么多省外的，结果最后去了青岛，很难评价。

迷宮飯

  1. 4/5
  2. #九井諒子

人物刻画细腻，美食制作丰富而且有一定思考。出圈作品。有时候看自己喜欢的作品动画化，然后爆火，心态很复杂。一方面开心作者有钱拿了，之后创作优秀作品的概率大大上升；另一方面有种看着自己喜欢的事物被一帮甚至可能没咋看过漫画出cos去漫展拍照交朋友恰大米的弱智蹭，也是蛮难受的。。。

黃金神威

  1. 5/5
  2. #野田サトル

冒险求生元素，同时展现风土人情，还有打猎等等元素，是不是穿插搞笑情节。剧情进行流畅，神作！问题是字多，不大适合kindle阅读，我第一遍在kindle上看到字多的地方就只看画面了，后来在平板上阅读体验好很多。插一嘴，拉大栓打猎真是太帅了。

作者在后期会插一些BL情节，为了搞笑？有点不明所以，但是不影响我评价这部作品为神作！

舞伎家的料理人

  1. 3.5/5
  2. #小山愛子

画风精美，而且有日本舞伎（md，这个字我输入法得翻好几页）成长的一些要素把。感觉算是中等偏上的美食漫画，舞姬（我不想打正确的了）成长的要素让美食漫画不至于千篇一律。但不至于像深夜食堂那种，每一话都有故事，压根不算美食漫画吧我认为，而且就算是有很多故事，看个几卷也疲劳了。

以后非必要不会出现这种3.5的评分，违背了我省事儿的初衷。但是这部漫画吧，确实也就是中等偏上水平，给3分可能有点委屈？但绝对到不了4分，2025年2月22日我看到第5卷。

涩涩

很多漫画中都会有涩涩情节，但是有些漫画涩涩才是主菜。你一眼就看出来作者就是要画涩涩，大威天龙，希望我的blog不至于因为这部分内容被封吧。。。

总之就先住民宿吧！

  1. 4/5
  2. #安藤優

作者就画了一卷，bangumi上评论反应停更了。但是真的所有元素都直击我的好球区！我一定要找到作者在海里的名字

恋爱

不知道该怎么写这部分的评语

大留工務店

  1. 5/5 优秀
  2. #山本周五郎 #望月ミネタロウ
  3. 日常漫画

望月好像是犬之岛的作者，犬之岛挺有名，好像还有电影，但我没看过。

我对这部漫画的评价是极为优秀，漫画体量不大，只有4卷。虽然说有点短，但是很理想。恋爱漫画长了就容易枯燥。我隐隐约约地感觉我看不下去高木同学就是这个原因，但是说实话高木同学得是好久之前看的了，已经不大记得了。不过我印象中，很多恋爱漫画我都是因为这种画日常感觉比较乏味枯燥，所以就没看下去过。感觉就像流水账一样。我印象中，像阿久津同学？长静同学不要欺负我？等等好几部。恋爱漫画感觉有的就是人物塑造不大行，作者把握不好这个，节奏拖沓，然后我就弃了。这一部只有四卷，虽然说略短，但是感觉比较好。

我感觉比较优秀的地方是人物塑造比较细腻，虽然说感觉有些地方（像是小孩偷东西那里，少东家抗压最后解释那部分），台词比较刻板，但是总体感觉人物塑造算是数一数二的细腻了，而且情节走的也比较好。所以我强烈推荐，给出满分。
