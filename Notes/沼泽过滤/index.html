沼泽过滤

发表于 2024-04-06 更新于 2024-07-27

历时一年的考研终于结束，仔细一想，差不多也是去年三月开始，今年也是在四月初迎来了结束。这一年，真是伤身体。不过现在终于有时间，发展下爱好里与计算机无关的方面，为家里阳台增添一些色彩，希望不是添一些麻烦

一些笔记








很多很多天以后。。。

我已经忘了补充笔记很久了，也忘了该从那里写了，唯一还留着的是我手机里的照片。先上图，这是今天（2024年7月26日）的状态。





过滤效果显著，水体清澈见底原谅我的拍摄技术，肉眼看不至于有这么大的反光，没有用物理过滤也达到了空气缸的效果。而且从沉淀仓的情况来看，沉淀仓非常有必要。

其实我一开始没有做沉淀仓的想法，刷视频刷到了，正好管子多，就用4分（外径20mm）的pvc管和弯头做了一个虹吸管，结果发现虹吸管小了！！反映出我物理水平属倒第一。然后我一气之下买了外径50的pvc管，在运输途中，我手痒难耐，挪了挪管子把水泵挪到了缸里，放弃了沉淀仓。效果也很好。

因此后来管子来的时候我很犹豫要不要加这个沉淀仓，因为我之前做的那个虹吸管是上pvc胶水的，很麻烦，味儿也挺大。最后买都买了我决定还是试试，由于我买的水管剪较小，没办法截50的管子。所以我在买管子的同时还买了个锯，然后就搁那儿吭哧吭哧地锯管子，与剪短20管时的轻松写意相比，相当之费力。

最后截好后懒得上胶水了因为装上弯头后拔都拔不出来，俺寻思估计也气密性不差，直接装好放缸里试了试，发现挺好。然后就留着了，第二天发现沉淀仓里很多龟便和火山石泥（？这里再提一嘴，真不该放火山石的，要不是有这个沉淀仓，我种水草时估计得种一回混一回。现在有了沉淀仓，泥能够沉淀到里边，种铜钱草和菖蒲还有绿九冠这几次翻动滤材，过滤里基本没有泥了，干净得很。

美中不足就是喂给龟的肉会被吸走

这个挂篮挂在龟缸里是临时之举，我买的垃圾桶做过滤，很符合我的预期。但是就是目前来说，水体对这俩挺水植物来说太深，没法种。我只能挂到边上等它们长一长。

再顺便一提，这个铜钱草长得是真快。我家阳台虽然采光还行，但是阳光直射还是少，但是这个铜钱草，到家我一开始没有篮子，直接扔到水管子上，第二天就开始有小芽，一天一个样，生长速度非常之迅速。相比之下，菖蒲的生长速度就了了。

除此之外，上图中有堵帽的那个，其实是我进水管的天线（？一开始我以为我这个过滤是竖着进水，不需要加防反流。事实证明是我物理水平低了，如果把不用这个天线通气，那进水管就是个虹吸管，水泵停电就会反流。不过这不是我加这个天线的主要目的，我一开始就想加这个天线，因为我检索到百度硝化细菌是好氧菌，我想用这个天线带下去空气。

构建过程

其实这部分该写到前边的。

选择垃圾桶就是选择了成功（bushi

我最后选择用垃圾桶做过滤有如下原因：

  1. 我最初并不是想把过滤放在旁边，而是想把过滤搭在龟缸上边。我养龟的这个周转箱一直用得还行，最初我是打算还找那家买的。结果那家不卖了周转箱了！！光是卖那种储物箱。那我寻思这种周转箱应该得是标准的把，我就跑pdd上看，结果很不幸地发现，挨，各家周转箱都是比较标准，尺寸也是比较统一。偏偏我已有的这个不是按标准来的，淦！怨不得它家不卖了我的这个周转箱的外部尺寸要比标准小一圈，不过内部尺寸是一致的。所以我估计，要是我真是往上摞箱子的话，影响应该也不大（？
  2. 不过同时周转箱还有一个问题，周转箱就那么几种尺寸，没大有很深的。因为我不打算用物理过滤了，所以我认为我的沼泽过滤滤材层要足够高，达到一定的物理过滤效果。
  3. 同时我去看了看成品的沼泽过滤箱，发现他们买的沼泽过滤箱是要比我买周转箱、水管、接头再自己做要便宜的！但是他们的沼泽过滤箱也有问题就是：沼泽过滤的部分不深；都做了滴滤；配置得较为复杂。我太清楚我自己了，我压根不可能给过滤换滤棉，也不会去定期打开排污口排污的。
  4. 综上所述，同时由于户外垃圾桶的价格，哪怕是相较于周转箱，都十分的便宜。我最终还是选择了使用垃圾桶做沼泽过滤箱。

更小的石英球。这里其实我本意是想买上图中那种较大的石英球来铺一层的，但是买的时候没看尺寸怨不得我说便宜这么多，到手了发现这么小！心痒难耐，就将就用了。

火山石和豆石

这么着就完成了，剩下的就是设置定时插座，连接水泵和沉淀仓，我没有拍照片。

我感觉是不应该放火山石的，粉化确实难以接受，如果没有沉淀仓，估计现在缸里会很浑。

安装好整个过滤后，就是得栽种植物。我的选择是：

  1. 铜钱草。长得嗷嗷快。
  2. 水菖蒲。emmmm，缸太深了，所以也现在没法种到滤材里。只能等它长一长。
  3. 绿九冠。水草，目前入缸3天，没啥变化。
  4. 垂泪莫斯。到手过完高锰酸钾后，就压石头下边了，不知是死是活。
  5. 一种水培的竹子？？我看家里这玩意儿长挺高，就拿了一根种缸里了，长势良好，起码比在花瓶的时候好。
  6. 一种芋类？？原因同4。
    后续计划加几条鱼进去。没想好加啥，本来觉得咖啡鼠挺好，但是一看是热带鱼，需要加热。我加热棒上全是水垢。看了看观背青鳉，兰寿（接受不来，恶心），黑鱼蟒纹黑龙。暂时还没想好。
