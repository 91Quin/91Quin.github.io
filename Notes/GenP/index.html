  • 新本子怎么办？我还得装个Acrobat不，现在的版本会不会有人投毒（我寻思这么多年了，该出事早出了会不会我没意识到
  • 去找Adobe的教育优惠。发现学生一年都要3000，淦！
  • 上v2ex找找看别的老哥怎么说。发现都在推荐pdf Xchange editor。
  • 去官网下载，没找到下载链接，只有第三方的站（我估计是我眼瞎或者ui缩放的锅，但我确实没找到。淦！无能狂怒。
  • 去52破解看看有没有文章，发现现在52不让发破解内容了。看到篇帖子提到vposy，我的超人，怎么把你给忘了
  • 回到微博，发现他微博删光了。一搜，停止更新。别的看着都是打着旗号的，不敢用。美好的互联网离我越来越远
  • 去找找m0nkrus，偶然发现r/GenP/。
    于是就开始这篇文章。

GenP是什么

A script patcher that changes some things inside the directory of the apps programs so that it extends trial periods for undefinite time / download it without attachments.

安全不?

总的来说，是通过管理控制的。emmm，都破解了，只能选个尽量安全的不是？

说实话我才刚开始看，我还没大理解。它这个好像是最开始是uncia开发的，但是2023年初停止维护了，更新到 GenP 3.0.3 (known in Reddit as GenP 3.0) ，同时也分享了源码。那后来就是八仙过海各显神通了。后面的项目，为了避免被修复起码reddit上是这么说的，就不再发布源码了。这些第三方的版本吧，乱七八糟，基本都带毒。emmmm，所以还是去选择reddit发布的版本，起码看上去安全。（md，我越写越觉得，这不就是不安全吗？嗯，基本上用破解也跟把自己电脑交给互联网上的一个陌生人差不多，看他良心呗用正版那就是看公司的良心了，说的就是你，XX/XXX/XXXX。从这一点讲，交给陌生人说不定还能放心点）

不过除了完全相信互联网上的好心人，还安装杀毒软件，或者就是分析一下文件等等。GenP的wiki也提到了:

You can also scan the program on VirusTotal and Hybrid Analysis.

嗯，你如果是个骗子，来骗我的时候一定要告诉我怎么识破你，那样我就会不带脑子地相信你，真的。

而且，考虑到我们现在正在进行的行为，杀毒软件一般都会报😅。

完了，死结了

没办法，都用破解了。所以最后还是需要靠自己我觉得这太裤辣，很符合我对赛博江湖的想象。

除此之外，你patch完可以把GenP删了，然后用杀毒软件扫描下你电脑。

If you wish, you can patch the programs and delete GenP from the PC after, as well scan your PC with Malware Bytes.
